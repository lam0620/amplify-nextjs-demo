1. Create nextjs project 
	npx create-next-app app-name
2. Initialize a new backend
	amplify init
3. Install Amplify Libraries
	npm install aws-amplify @aws-amplify/adapter-nextjs
4. Build an Authentication experience using the Authenticator
	npm install aws-amplify @aws-amplify/ui-react
